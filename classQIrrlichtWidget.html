Detailed Description

Irrlicht Engine Widget.

This class can be used to display a Irrlicht 3D Engine powered scene in your Qt application.

Author
jcfalcone (http://irrlicht.sourceforge.net/forum/viewtopic.php?f=1&t=48694)
Sebastian Schmidt
